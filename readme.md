# totp

A simple CLI tool to store 2FA tokens and retrieve time based OTPs when needed

### Usage

```bash
Usage: totp <COMMAND>

Commands:
  add     
  remove  
  list    
  help    Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```

### Building

```bash
cargo build --release
```
